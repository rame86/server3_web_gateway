## 아이디어 A: Neo-Maximalist K-Wave

**Design Movement:** Neo-Maximalism + Korean Wave Aesthetic (한류 미학)

**Core Principles:**
1. 화려하고 대담한 색상 대비 — 검정 배경 위에 형광 핑크/퍼플/시안이 폭발적으로 터지는 느낌
2. 레이어드 정보 구조 — 카드와 패널이 겹쳐지는 깊이감 있는 레이아웃
3. 아티스트 중심 시각 언어 — 아티스트 이미지가 UI를 지배하는 포스터 스타일
4. 디지털 글리치 + 홀로그램 효과 — 미래지향적 K-POP 비주얼 코드

**Color Philosophy:**
- 배경: 딥 블랙 `#0A0A0F`
- 주 강조: 형광 핫핑크 `oklch(0.65 0.35 340)`
- 보조 강조: 일렉트릭 퍼플 `oklch(0.55 0.30 290)`
- 하이라이트: 네온 시안 `oklch(0.85 0.20 195)`
- 감성: "무대 위의 스포트라이트" — 어둠 속에서 빛나는 아티스트

**Layout Paradigm:**
- 비대칭 매거진 레이아웃 — 좌측 큰 아티스트 이미지 + 우측 정보 스택
- 풀스크린 히어로 섹션에 아티스트 사진 오버레이
- 사이드바 네비게이션 (아이콘 + 텍스트)

**Signature Elements:**
1. 그라디언트 글로우 버튼 — 핑크→퍼플 그라디언트 + 호버 시 외부 글로우
2. 홀로그래픽 카드 — 마우스 무브 시 무지개빛 반사 효과
3. 타이포그래피 믹스 — 굵은 한글 + 영문 대문자 조합

**Interaction Philosophy:**
- 모든 인터랙션에 파티클/글로우 피드백
- 스크롤 시 패럴랙스 + 요소 페이드인
- 호버 시 카드가 살짝 떠오르는 3D 틸트

**Animation:**
- 입장 애니메이션: 슬라이드업 + 페이드인 (stagger 0.1s)
- 배경: 천천히 움직이는 그라디언트 메시
- 버튼 클릭: 리플 + 글로우 펄스

**Typography System:**
- 헤딩: Noto Sans KR Bold 900 + 영문 Bebas Neue
- 본문: Noto Sans KR Regular 400
- 강조: 이탤릭 + 핑크 컬러

## 아이디어 B: Soft Bloom — 파스텔 팬덤 감성 ✅ 선택

**Design Movement:** Contemporary K-Beauty Aesthetic + Soft Editorial

**Core Principles:**
1. 따뜻하고 화사한 파스텔 팔레트 — 로즈, 라벤더, 피치, 민트의 조화
2. 플루이드 유기적 형태 — 직각 대신 물결치는 곡선과 블롭 형태
3. 레이어드 글래스모피즘 — 반투명 카드와 부드러운 블러 효과
4. 아티스트 중심 스토리텔링 — 팬과 아티스트의 감정적 연결을 시각화

**Color Philosophy:**
- 배경: 크리미 화이트 `oklch(0.98 0.01 60)`
- 주 강조: 딥 로즈 `oklch(0.60 0.20 10)`
- 보조 강조: 소프트 라벤더 `oklch(0.75 0.12 290)`
- 액센트: 피치 코랄 `oklch(0.80 0.15 30)`
- 다크 텍스트: 딥 플럼 `oklch(0.25 0.08 320)`
- 감성: "꽃이 피는 봄날, 아티스트와 팬의 설레는 만남"

**Layout Paradigm:**
- 비대칭 매거진 그리드 — 큰 히어로 이미지와 작은 카드들의 리듬감 있는 배치
- 좌측 고정 사이드바 (아이콘 + 레이블) + 우측 콘텐츠 영역
- 섹션 구분에 물결 SVG 디바이더 사용

**Signature Elements:**
1. 블롭 배경 장식 — 페이지 곳곳에 반투명 블롭 형태의 배경 요소
2. 글래스 카드 — `backdrop-blur` + `bg-white/60` + 소프트 섀도우
3. 그라디언트 텍스트 — 로즈→라벤더 그라디언트 헤딩

**Interaction Philosophy:**
- 부드럽고 우아한 트랜지션 — cubic-bezier 이징
- 호버 시 카드가 살짝 위로 뜨며 섀도우 깊어짐
- 클릭 시 잔물결 효과 (ripple)

**Animation:**
- 입장: 아래에서 위로 부드럽게 페이드인 (stagger 0.08s)
- 배경 블롭: 천천히 떠다니는 애니메이션 (20s loop)
- 페이지 전환: 크로스페이드

**Typography System:**
- 헤딩: Playfair Display (영문) + Noto Serif KR (한글)
- 본문: Noto Sans KR 400/500
- 강조 숫자: DM Sans Bold

**이유:** K-POP 팬덤의 따뜻하고 설레는 감성을 가장 잘 표현하며, 다양한 연령대의 팬들에게 친근하고 화사한 인상을 줍니다. 글래스모피즘과 파스텔 팔레트의 조합은 트렌디하면서도 가독성이 높아 정보량이 많은 플랫폼에 적합합니다.
